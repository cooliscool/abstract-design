|ABOUT THE PROJECT

Designing for an event organising company was exciting yet tiring at the same time. The application required two different log ins; one for the event company and the other for public to book events. The log in for the event organiser had to be more functional oriented rather than the UI, where as the user log ins had to have the best of both worlds.
